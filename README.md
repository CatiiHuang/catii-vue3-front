# 基于Vite+Vue3的项目快速生成工具  catii-vue3-front 
**可快速生成项目模板**
`usage`:
`npm create catii-vue3-front `
`npm create catii-vue3-front  projectName`
### 已配置模块
- Vite + Vue3 + VueRouter + Vuex
- ElementPlus + Less + axios
- Eslint + Prettier + eslint-config-tencent
